# schedule-1-poptracker-razgriz
A PopTracker for Schedule I

This is my first PopTracker so don't expect anything fancy. I did the best I could with the limited skills I have.

There are probably logic rules I misunderstood or randomization option combinations that I messed up on.

You can ping me on Discord (I am in the Archipelago After Dark server as Razgriz) to let me know of any errors.

TO USE: Simply download the .zip file and put it in your Poptracker packs folder. It should work then.

KNOWN LIMITATIONS: I do not yet know how to add offline settings to a PopTracker.
When you load it, it will display everything as green because it things all settings are non-randomized so everything is accessible.
However, when you connect it to an Archipelago server, it will access your slot data and adjust accordingly.
